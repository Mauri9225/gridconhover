“Grid con hover explicativo”
Instrucciones:
1. Crea una matriz fija de 5 x 5 celdas numeradas del 1 al 25.
2. Cuando el usuario pasa el mouse (hover) sobre una celda:
    - Muestra debajo de la matriz un texto que diga:
    * “Número: X”
    * “Es par / impar”
    * “Es múltiplo de 3: sí / no”
3. Usa CSS para:
    - Aplicar un efecto visual al :hover (por ejemplo cambio de color y un transform: scale(1.05)).
    - Agregar transition para que la escala y el color cambien suavemente.
4. Toda la lógica de par/impar y múltiplo de 3 debe hacerse en JavaScript, usando el número de la celda.
